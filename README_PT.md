# Unified Portal — PoC Técnico

**Autor:** Manuel Silva (2025)

## Visão Geral
Este repositório contém um PoC para um Portal Unificado que centraliza o acesso a ferramentas internas e facilita o onboarding e a segurança. Inclui:
- Frontend Next.js (PoC)
- Handler de protocolo `portal://` e launcher local
- Exemplos de integração Azure AD (NextAuth)
- Guia para distribuição via Intune
- Documentação e materiais de demo

## Quickstart (dev)
1. `cd Portal_NextJS`
2. `npm install`
3. `npm run dev`
4. Aceder a http://localhost:3000 e clicar 'Login SSO' (mock) e testar os cartões de apps.
